# 나만의 음성 비서 (Voice Assistant)

이 프로젝트는 OpenAI API(Whisper 및 GPT)와 Streamlit을 활용하여 만든 웹 기반 음성 비서 애플리케이션입니다. 사용자의 음성을 텍스트로 변환(STT)하고, GPT 모델을 통해 답변을 생성한 뒤, 다시 음성으로 들려주는(TTS) 기능을 갖추고 있습니다.

## 주요 기능

- **음성 인식 (STT)**: OpenAI Whisper API를 사용하여 사용자의 한국어/영어 음성을 정확하게 텍스트로 추출합니다.
- **지능형 대화 (Chatbot)**: OpenAI GPT-4 또는 GPT-3.5-turbo 모델을 사용하여 사용자의 질문에 답변합니다.
- **음성 합성 (TTS)**: Google Translate TTS(gTTS)를 사용하여 생성된 답변을 자연스러운 한국어 음성으로 출력합니다.
- **채팅 UI**: Streamlit을 활용한 직관적인 인터페이스와 아이폰 스타일의 말풍선 디자인을 제공합니다.
- **세션 관리**: 대화 기록이 유지되어 이전 대화 내용을 바탕으로 답변이 가능합니다.

## 설치 및 실행 방법

### 1. 사전 요구 사항

- Python 3.8 이상 설치
- OpenAI API Key 보유
- (선택 사항) 오디오 처리를 위한 `ffmpeg` 설치

### 2. 패키지 설치

터미널에서 아래 명령어를 입력하여 필요한 라이브러리를 설치합니다.

```bash
pip install -r requirements.txt
```

### 3. 애플리케이션 실행

아래 명령어를 입력하여 Streamlit 서버를 실행합니다.

```bash
streamlit run app.py
```

실행 후 브라우저에서 `http://localhost:8501` 주소로 접속하면 사용할 수 있습니다.

## 사용법

1. 사이드바의 **OPENAI API 키** 입력란에 개인 API 키를 입력합니다.
2. 사용할 **GPT 모델**을 선택합니다.
3. **"클릭하여 녹음하기"** 버튼을 누르고 음성을 녹음합니다.
4. 녹음이 완료되면 AI가 질문을 분석하고 음성으로 답변합니다.
5. 대화를 초기화하고 싶다면 사이드바의 **"초기화"** 버튼을 클릭합니다.

## 프로젝트 구조

- `app.py`: 메인 애플리케이션 소스 코드
- `requirements.txt`: 설치가 필요한 관련 패키지 목록
- `README.md`: 프로젝트 설명서

## 주의 사항

- OpenAI API 사용 시 비용이 발생할 수 있습니다.
- `ffmpeg`가 설치되어 있지 않을 경우, 일부 환경에서 오디오 녹음 및 재생이 원할하지 않을 수 있습니다.
